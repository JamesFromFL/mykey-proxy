# mykey-biometrics

Planned home for MyKey biometric setup and management.

This module will eventually provide:

- backend selection for `fprintd` and `Howdy`
- enrollment setup flows
- backend validation and test flows
- MyKey-managed biometric policy enablement
- orchestration of existing system biometric stacks rather than a reimplementation of biometric verification inside MyKey

It is intentionally a placeholder for now while `mykey-pam` Phase A and the
existing `mykey-pin` backend continue to stabilize.
